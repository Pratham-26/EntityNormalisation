# Entity Normalisation

Entity normalisation project built with Zig.

## Overview

This project focuses on normalising entities across different data sources.

## Getting Started

### Prerequisites

- Zig 0.11+

### Installation

```bash
git clone https://github.com/Pratham-26/EntityNormalisation.git
cd EntityNormalisation
zig build
```

## Usage

```bash
zig build run
```
